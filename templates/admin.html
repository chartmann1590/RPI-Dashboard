API Configuration Required

To enable weather and news features, you need to obtain API keys:

1. Weather API: Get a free key from OpenWeatherMap

2. News API: Get a free key from NewsAPI.org

Update the keys in app.py: WEATHER_API_KEY and NEWS_API_KEY
